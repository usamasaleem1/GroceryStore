# What
In this website, you can view grocery items, add items to cart, checkout, and sign up/sign in to save your progress, and more.

# Why
This grocery store project was created as part of a semester long project. 

# Stack
- HTML
- JavaScript
- PHP
- MySQL
- Heroku
- Github Hosting
